# GenAI Recommender Journey (Weeks 1-4)

This repository documents the development of a Generative AI-powered Recommender System, spanning from initial data preparation to hybrid ranking models.

## Project Overview
The goal of this project is to build an intelligent recommendation engine that leverages traditional collaborative filtering techniques alongside modern Generative AI and embedding-based retrieval to provide highly relevant product suggestions.

## Weekly Progress Summary

### [Week 1: Foundation & Data Engineering](./GenAI_Recommender_DynamoB_Sprint1_Week1/)
*   **Focus:** Environment setup and data pipeline.
*   **Key Activities:** Synthetic data generation for products and user behavior, data cleaning, and initial exploratory data analysis (EDA).

### [Week 2: Baseline Recommender Models](./GenAI_Recommender_DynamoB_Sprint1_Week2/)
*   **Focus:** Implementing traditional recommendation algorithms.
*   **Key Activities:** Developed Popularity-based, Collaborative Filtering, and Matrix Factorization models to establish performance baselines.

### [Week 3: Content-Based Filtering & Embeddings](./GenAI_Recommender_DyanamoB_Sprint2_Week3/)
*   **Focus:** Deep learning and vector representations.
*   **Key Activities:** Generated product embeddings using LLMs, implemented a content-based recommender, and integrated a FAISS vector database for efficient similarity search.

### [Week 4: Hybrid Recommender & Learning to Rank (LTR)](./GenAI_Recommender_TeamDynamoB_Sprint2_Week4%20(2)/)
*   **Focus:** Model fusion and optimization.
*   **Key Activities:** Integrated collaborative and content-based signals into a Hybrid Recommender. Implemented a Learning to Rank (LTR) model to optimize the final recommendation list.

*Note: This repository currently contains progress up to Week 4. Data files and model checkpoints are excluded for security and efficiency.*
